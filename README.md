# 校园维修系统

## 技术运用

Java mybatis框架，MySQL数据库，使用Maven进行项目管理。

## 功能需求
**公共功能**

1. 用户注册（选择角色，学号/工号需正则校验）
2. 用户登录
3. 修改密码
4. 查看自己的基本信息

**学生功能**

1. 绑定宿舍：首次登录后绑定宿舍楼栋和房间号
2. 创建报修单：选择设备类型，填写问题描述，自行设计报修单状态（如：“已完成”）
3. 查看自己的报修记录：列表展示，可查看详情
4. 取消报修单

**管理员功能**

1. 查询所有报修单（支持按状态筛选）
2. 查看单个报修单详情
3. 修改报修单状态（并更新最后修改时间）
4. 删除报修单
